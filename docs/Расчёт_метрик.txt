1. Метрики операционной эффективности
	1. Time Reduction % (Снижение времени андеррайтинга на одного клиента)
Система автоматического вычисления прогноза снижает время, затраченное на автоматическое выполнение прогноза, по сравнению с ручным способом вычисления стоимости страхования.
Time Reduction % = (Базовое время - Новое время) / Базовое время × 100%
Базовые расходы (в среднем) = 60 мин
Новые расходы (в среднем) = 10 мин
Time Reduction % = 83% снижения затра по времени
	2. Cost Saved per Month (экономия в рублях за месяц)
Cost Saved per Month = (Базовое время - Новое время) × Часовая ставка аналитика × Количество прогнозов
Cost Saved per Month = 270000 руб/мес (на 500 прогнозов/мес)
	3. Cost per Inference
Систему необходимо развернуть на сервере, чтобы удалённо выполнять запросы.
Cost per Inference = (Затраты на инфраструктуру модели за период) / (Количество инференсов за период)
Затраты на инфраструктуру модели за период = 15000 руб/мес (аренда в облаке)
Количество инференсов за период = 3000 в мес
Cost per Inference = 5 руб.
	4. Return on Investment (ROI)
Показывает окупаемость за месяц
ROI (месячный) = (Cost Saved - OPEX) / OPEX × 100%
OPEX (Operating Expenses) - текущие операционные расходы на эксплуатацию ИИ-системы
OPEX = 65000 руб/мес (аренда сервера + поддержка)
ROI (месячный) = 315 %

2. Архитектурные метрики надежности
	1. Latency (время отклика) 
Время ответа системы на запрос расчёта предсказания прогноза.
Latency P95 = 57,237 мсек. (10000 запросов на локальный сервер)
Для удалённого сервера Latency P95 ~= 200 мсек.
	2. Throughput (Пропускная способность)
Сколько операций система обрабатывает в единицу времени.
Throughput = 14 прогнозов/сек.
	3. API Error Rate (частота ошибок API)
Отношение числа неспешных запросов к числу успешных.
API Error Rate = 0,03 %
	4. Initial Load Time (время начальной загрузки)
Initial Load Time P95 = 2,23 сек.
	5. Среднее время распознавания анкеты OCR
Time OCR = 3,51 сек
